# upublish (Claude Code plugin)

Publish static sites to the web instantly via Claude Code.

## Install

```sh
/plugin install omni-ping/upublish.claude
```

Or via npx:

```sh
npx skills add omni-ping/upublish.claude
```

## Usage

1. `/upublish:setup` — installs CLI, configures MCP, authenticates
2. `/upublish:ask` — publish, list, delete, manage sites

## Other platforms

- **Codex / Gemini / others:** Use [upublish.skill](https://github.com/omni-ping/upublish.skill)
- **Standalone CLI:** `npm install -g @upublish/cli`
